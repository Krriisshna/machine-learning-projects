# House Price Prediction
## Objective
Predicting house price using features like:
- area
- rooms
- bathrooms
## Dataset
Kaggle House Prices - Advanced Regression Techniques competition
## Techniques Used
Decision Tree and Random Forests
## Result
- Decision Tree MAE: 33632
- Random Forests MAE: 35781
## Conclusion
Decision Tree performed better in this experiment.  
This may be because only three features were used and the Random Forest model was not further tuned.
